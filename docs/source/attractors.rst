`attractor` Module
---------------------------

This is the main module which contains all methods necessary for generation of the attractor
coordinates and also the visualization of those attractors.

`parser` Module
------------------------

This is the module which wraps the :mod:`attractors.attractor` module,
and provides an easy to use CLI. As of ``v1.3.0``, the parser module only supports animations.
